# Network Behaviour Monitor

Network Behaviour Monitor is a local defensive cybersecurity project that observes network traffic and converts packet data into readable network flows.

The project aims to make network activity easier to understand by grouping traffic based on source IP, destination IP, protocol, port number, and packet size.

It will later include baseline-based behaviour analysis, basic risk scoring, and plain-language alerts for unusual activity.

## Current Status

Project setup started.

## Planned Features

- Local packet/log input
- Flow generation
- Behaviour baseline
- Risk scoring
- Alert explanations
- Simple dashboard or readable output

## Ethical Scope

This project is local and defensive only. It will not perform external scanning, exploitation, credential attacks, or offensive activity.
